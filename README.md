# RTL React Starter

A reusable React starter template built with Vite, Tailwind CSS, and React Router. Designed for RTL (Right-to-Left) applications and organized with a scalable folder structure for faster project setup.

## Features

* ⚡ Vite
* ⚛️ React
* 🎨 Tailwind CSS
* 🛣️ React Router
* 🌍 RTL-ready
* 📁 Scalable project structure
* 🧩 Reusable components and layouts
* 🛠️ Utility-first development workflow

## Tech Stack

* React
* Vite
* Tailwind CSS
* React Router

## Installation

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Project Structure

```text
src/
├── assets/        # Images, icons, fonts, and static files
├── components/    # Reusable UI components
├── hooks/         # Custom React hooks
├── layouts/       # Application layouts
├── pages/         # Route pages/screens
├── App.jsx
├── main.jsx
└── index.css
```

## RTL Support

This starter is intended for RTL applications and provides a clean foundation for building Arabic and other right-to-left web applications.

## Usage

Use this template as a starting point for new projects:

1. Clone or use the repository as a template.
2. Install dependencies.
3. Add your pages, components, and routes.
4. Start building.
